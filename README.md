1.	Вимоги до середовища
Для запуску проєкту необхідно:
•	Node.js (версія ≥ 16);
•	npm або yarn для керування залежностями;
•	Expo CLI для швидкого запуску та тестування;
•	Firebase акаунт для автентифікації користувачів;
•	Мобільний пристрій або емулятор Android/iOS.

2.	Інструкція запуску
Клонування репозиторію:
Для отримання вихідного коду проєкту необхідно виконати команди:
git clone https://github.com/Judy21fei/Course-work-third-year-.git
cd Course-work-third-year-
Ці команди:
•	завантажують копію репозиторію з GitHub на ваш комп’ютер;
•	переходять у директорію проєкту для подальшої роботи.
Встановлення залежностей:
npm install
Налаштування Firebase
•	Створити проєкт у Firebase Console.
•	Додати конфігурацію у файл firebaseConfig.js.
•	Переконатися, що активовано модуль Authentication (Email/Password).
Запуск застосунку:
npx expo start
Вибрати запуск на Android/iOS емуляторі або реальному пристрої через Expo Go.

3. Структура файлів проєкту:
•	App.js – головний файл застосунку. Містить маршрутизацію між екранами та базову логіку.
•	firebaseConfig.js – конфігурація Firebase. Використовується для автентифікації користувачів.
•	RegistrationScreen.js – екран реєстрації нового користувача.
o	Поля вводу: email, пароль, підтвердження пароля.
o	Валідація даних та повідомлення про помилки.
o	Інтеграція з Firebase для створення облікового запису.
•	LoginScreen.js – екран входу користувача.
o	Поля вводу: email, пароль.
o	Авторизація через Firebase.
o	Повідомлення про помилки у доступній формі.
o	Перехід до реєстрації у випадку відсутності акаунта.
•	HomeScreen.js – головна сторінка застосунку.
o	Масштабування тексту (функції increaseFont, decreaseFont).
o	Перемикання контрастного режиму (toggleContrast).
o	Навігація через кнопки Explore Services та Learn More.
o	Фіксований заголовок (header) з доступними кнопками.
•	Стилі у кожному файлі
o	Єдина система оформлення інтерфейсу.
o	Контрастні кольори, прості шрифти, великі кнопки.
o	Адаптивність для різних розмірів екранів.
•	assets/ – папка з графічними ресурсами (іконки, зображення).
